
jQuery 


jQuery  核心
	是什么：
		轻量的开源 js 库；简化 HTML 和 JavaScript 之间的操作；
	为什么使用：
		文档全面；插件丰富；跨主流浏览器。
	基本原理：
		封装了 JavaScript 的部分内容。
	怎么使用：
		导入 jQuery 库；
		<head> 标签中导入 jQuery.js 文件；
		调用 jQuery 的 API；
			也允许使用 js 语法。
		js 对象：
			innerHTML
			value
			removeChild()
			appendChild()
		jQuery 对象：
			html()
			val()
			remove()
			append()
