编程等宽字体

Consolas

发行：Microsoft


Consolas 雅黑混合字体

Consolas + 中文雅黑混合版（YaHei Consolas Hybrid）
是最为适宜的字体选择。
相对Source Code Pro字体而言，宽度适中，
而宋体的字号在<12磅时中文字体明显不如雅黑舒服，
而其它字体则在中文字体支持上，
以及"l1O0"(注意是小L和数字1，大O和数字0)差别上分别有不足。



Courier New

发行：IBM


Droid Sans Mono

Droid Sans Mono 是 Google Droid 字体家族中一员,
专门用来开发 Android 平台上的应用。
大小为 10 pt 时，字符间距很合适。
但是如果你使用较小分辨率的屏幕的话（我现在使用的是1280×800），
采用 9 pt 能在屏幕上显示更多代码。


Source Code Pro

Adobe 在发布首款开源字体 Source Sans Pro 后，
2012年9月又发布了 Source Code Pro 字体集，
它是一款非常适合程序员阅读的等宽字体，
适用于 Linux/Unix、Mac OS X 和 Windows 系统，
由 Paul D. Hunt 设计，遵循 SIL Open Font 协议。


Liberation Mono 

因其较高的可读性，是程序员的一个不错的选择。
