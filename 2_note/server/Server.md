### 磁盘的数据如何存储和读取

视频讲解：[1. mysql 面试题 - 深入理解 B + 树原理](https://www.bilibili.com/video/BV15V411p7pi)

磁盘内部分为多个盘片。

盘片上有多个磁道，是均匀分布的同心圆，由磁盘中心中心向外扩散。

盘片旋转的时候，磁头滑向对应的磁道读取数据。

单个磁道可以看作是厚度很薄的圆环，圆环被均匀分成多段弧形，这些相对独立的弧形叫扇区。

磁盘最小的存储单位扇区，容量是 512 M。
