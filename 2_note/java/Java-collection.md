#### 基础部分

###### Set 接口的实现

HashSet：

    根据元素的 HashCode 做去重

TreeSet：

    根据元素的 HashCode 做去重，并用平衡树进行排序

###### Queue 接口的实现

PriorityQueue：

    根据元素排序决定出列的顺序
