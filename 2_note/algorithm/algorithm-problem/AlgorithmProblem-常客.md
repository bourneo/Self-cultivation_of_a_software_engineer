### Java 数据结构

ArrayDeque：

    是双端队列的实现类；
    
    其实现的接口 Deque 接口中定义了双端队列的主要的方法，比如从头删除，从尾部删除，获取头数据，获取尾部数据等等。
    
    deque 是 “double ended queue（双端队列）” 的缩写。
    
    可以用来做队列和栈。
